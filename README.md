Этот сервис позволяет создавать события, регистрировать пользователей и бронировать места на события.  
Реализован на Go с использованием PostgreSQL для хранения данных.

## Возможности

- Создание, получение и список событий  
- Создание и список пользователей  
- Создание бронирований  
- Получение списка участников события

## Структура базы данных

В проекте используется следующая схема БД (описана в `schema.sql`):

- Таблица `events` — события  
- Таблица `users` — пользователи  
- Таблица `bookings` — бронирования

## Использование

Для взаимодействия с API используйте HTTP-запросы по эндпоинтам:

- POST /users - создать пользователя
- POST /events - создать событие
- POST /bookings - забронировать место
- GET /events/{id}/participants - список участников события
